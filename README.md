# Godot Jump Game

Yksinkertainen 2D-hyppelypeli Godot 4:llä.

## Ominaisuudet

- 🎮 Pelaaja voi liikkua vasemmalle/oikealle nuolinäppäimillä
- ⬆️ Hyppääminen välilyönnillä
- 🎯 Alustat joilla liikkua
- 📷 Kamera seuraa pelaajaa
- 💥 Pelin resetointi kun pelaaja putoaa

## Ohjeet

1. Avaa Godot 4 ja avaa tämä projekti
2. Paina "Play" (F5) käynnistääksesi pelin
3. Käytä nuolia liikkumiseen ja välilyöntiä hyppäämiseen
4. Yritä päästä korkeimmalle!

## Rakenne

```
├── project.godot        # Projektin konfiguraatio
├── scripts/
│   ├── Player.gd       # Pelaajan logiikka
│   └── Level.gd        # Tason logiikka
└── scenes/
    └── main.tscn       # Pääscene
```
